# Numbskull

## Installation
#### Ubuntu 14.04
```sudo apt install libboost-all-dev cmake```

This will install boost 1.54.0 and openmpi 1.6.5

#### Ubuntu 16.04
```sudo apt install libboost-all-dev cmake```

This will install boost 1.58.0 and openmpi 1.10

## Compile and Run
```
cd make-build-debug
make run
```
