# Numbskull
A distributed Gibbs smapling framework
Still developing...

## Installation Dependency
### Ubuntu 16.04
```bash
sudo apt update
sudo apt upgrade
sudo apt install libboost-all-dev cmake
```
These commands will install boost 1.58.0, openmpi 1.10.2 and cmake 3.5.1
